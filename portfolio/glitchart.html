G L I T C H   A R T

These series of glitch art display three paintings based on Christianity with each showing the three seperate planes of existence: Heaven, Earth, and Hell. Respectively, the Creation of Adam represents Heaven, the Crucifixion of Jesus represents Earth, and the Fallen Angel represents Hell. These glitches represents the idea that religion as a concept in society is sort of like a glitch too.

Creation of Adam

This glitch was made using the text editor, Notepad++. I put the first two chapters of the book of Genesis into the code which resulted with each chapter creating a section of red throughout the image.

Crucifixion of Jesus

This glitch was also made using the text editor, Notepad++. Similar to what I did with the Creation of Adam, I put the parts in which Jesus was crucified in the Bible, four times. Each time was from the four different books, Matthew, Mark, Luke, and John, which were four different accounts on the life of Jesus.

Fallen Angel

This glitch made using the audio editor, Audacity. I turned the original image of the painting into a RAW file and changed its high pass filter into 666, thus creathing this glitched rainbow effect with a burning sky.
